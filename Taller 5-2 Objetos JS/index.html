1) Mostrar solo los valores

2) Mostrar solo las llaves

Objeto base: { v1:"Tomate", v2:"Cebolla", v3:"Ajo" }

3) Mostrar llaves y valores

Imprime pares clave→valor del objeto base.

4) Agregar 2 verduras y mostrar valores

Se agregan Zanahoria y Lechuga al objeto.

5) Productos dinámicos (mín. 3)

Objeto: { nombre, precio, cantidad }. Agrega productos y muéstralos.

Agrega al menos 3 productos para ver la tabla.

6) Productos dinámicos – Versión array de objetos

Mismo objetivo del 5, mostrando otra estructura.

7) Subtotales y total con IVA

Objeto: { producto, precio, cantidad } (mín. 2). Calcula subtotales, total y total con IVA.

Agrega productos (mín. 2). Si hay 3 o más, se mostrará también el subtotal de los primeros 3.
